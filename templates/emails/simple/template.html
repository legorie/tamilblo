The Template with a variable of {{variable}}

Template interpolation for Mandrill:

  • name *|NAME|*
  • first_name *|FIRST_NAME|*
  • email *|EMAIL|*
  • testVar *|TESTVAR|*

    • Template interpolation for Mailgun:

  • name {{ name }}
  • firstName {{ firstName }}
  • email {{ email }}
  • testVar {{ testVar }}
